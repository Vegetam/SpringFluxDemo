# SpringWebFluxDemo
A demo of using Spring Flux to build REST API

#Technology Stack

Java 17

Spring Boot 3

Spring WebFlux (Reactive Programming in Java)

Lombok

Maven

MongoDb

Swagger

Junit 5

# How to use ?

mvn clean install

mvn spring-boot:run
